# EntglDb Documentation

This website is built with [VitePress](https://vitepress.dev).

## Run Locally

```bash
# Install dependencies
npm install

# Start development server
npm run docs:dev

# Build for production
npm run docs:build
```

## Structure
- `index.md`: Home page
- `getting-started.md`: Guide
- `architecture.md`: Concepts & Security
- `api-reference.md`: API Docs
